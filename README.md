# 🌐 PostUp - Social Posting Platform

A full-stack social media application built using **Node.js, Express, MongoDB, EJS, Cloudinary, and Day.js**.

PostUp allows users to create accounts, share posts, interact with content, manage profiles, and explore other users through a responsive and modern social platform experience.

## 🌐 Live Demo

https://postup.onrender.com/

---

## ✨ Features

- User authentication and authorization
- Secure signup, login and logout system
- Create, edit, and delete posts
- Like functionality for posts
- Personalized user profiles
- Profile image upload support
- Public profile viewing
- User search functionality
- Feed with latest posts displayed first
- Joined date, post count, and profile stats
- Human-readable date & time formatting using Day.js
- Fully responsive interface across devices
- Persistent MongoDB database integration
